## 📌 Project Title
**DecodeLabs Project 2 — Backend API Development**

---

## 📖 Description

A fully functional **RESTful Backend API** built with **Node.js + Express**.

It handles a **User Management System** with full CRUD operations — Create, Read, Update, and Delete users. The API validates all incoming data, returns proper HTTP status codes, and handles errors gracefully.

**What it does:**
- `GET /users` — Returns list of all users
- `GET /users/:id` — Returns a single user by ID
- `POST /users` — Creates a new user (with validation)
- `PUT /users/:id` — Updates an existing user
- `DELETE /users/:id` — Deletes a user

---

## ▶️ How to Run

**Step 1 — Install dependencies**
```bash
npm install
```

**Step 2 — Start the server**
```bash
node server.js
```

**Step 3 — Open in browser**
```
http://localhost:3000
```
